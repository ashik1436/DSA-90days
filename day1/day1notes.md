# Day 1: Two Sum & Reverse String

## What I Learned
1. **Hash Map:** Store values for quick lookup
2. **Two-Pointer:** Use pointers from both ends to solve problems

## Two Sum
- Problem: Find two numbers that add to target
- Solution: Use dictionary to remember numbers
- Time: O(n) - one loop
- Space: O(n) - storing numbers

## Reverse String
- Problem: Reverse array in-place
- Solution: Swap from both ends
- Time: O(n) - visit each element once
- Space: O(1) - no extra array
